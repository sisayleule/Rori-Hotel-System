Rori Hotel Photos Directory Setup
---------------------------------

Please copy your Rori Hotel photos into these folders with these exact file names:

1. Inside the 'hero' folder:
   - hero1.jpg: Your best wide exterior shot or main lobby photo. Used as the full-screen background for the hero section.
   - Recommended minimum width: 1200 pixels.

2. Inside the 'reception' folder:
   - reception1.jpg: Used in the full-screen scroll gallery.
   - reception2.jpg: Used in the department card.
   - Recommended minimum width: 800 pixels for cards.

3. Inside the 'rooms' folder:
   - rooms1.jpg: Used in the full-screen scroll gallery.
   - rooms2.jpg: Used in the department card.
   - Recommended minimum width: 800 pixels.

4. Inside the 'kitchen' folder:
   - kitchen1.jpg: Used in the full-screen scroll gallery.
   - kitchen2.jpg: Used in the department card.
   - Recommended minimum width: 800 pixels.

5. Inside the 'fb' folder:
   - fb1.jpg: Used in the full-screen scroll gallery.
   - fb2.jpg: Used in the department card.
   - Recommended minimum width: 800 pixels.

Note:
- The website uses CSS 'object-fit: cover' which will automatically crop all photos to fill their layouts perfectly without distortion, so you do not need to manually resize them.
- Until real Rori Hotel photos are added in these directories, the homepage/website will show broken image icons, but all core application flows and login systems will continue working correctly! Feel free to add files at any time.
