## 🎨 Key Visual Elements

### 1. Double Border Frame
- **Outer**: 4pt gold border with 15pt rounded corners
- **Inner**: 1pt dark gold border with 8pt rounded corners
- **Effect**: Creates luxury frame, draws eye to content
- **Spacing**: 20pt from page edge (outer), 35pt (inner)

### 2. Top & Bottom Medallions
- **Structure**: Center circle with two horizontal accent lines
- **Circle**: 25pt radius outer, 20pt radius inner (double ring)
- **Lines**: 3pt thick gold lines extending left and right with 80pt gap
- **Purpose**: Decorative elements that balance top and bottom
- **Customization**: Perfect space for hotel logo in center circle

### 3. Student Name Treatment ⭐
- **Font**: 46pt Helvetica-Bold (largest text on certificate)
- **Color**: Elegant gold (#C9A84C) for prominence
- **Underline**: 250pt long, 2.5pt thick gold line
- **Ornaments**: Two 3pt gold circles on ends of underline (8pt away)
- **Effect**: Makes the certificate personal and prestigious

### 4. Performance Card
- **Background**: Warm cream color (#FFF8E7)
- **Border**: 2pt gold border
- **Accent Bars**: 6pt wide gold bars on left and right edges
- **Height**: 85pt tall
- **Padding**: 110pt from left/right edges
- **Content**: Three columns with large numbers and labels

### 5. Typography Hierarchy
```
Level 1: Student Name (46pt Gold)     ← HERO
Level 2: Hotel Name (38pt Dark)       ← BRAND
Level 3: Performance Numbers (32pt Gold) ← DATA
Level 4: Title (28pt Dark)            ← CONTEXT
Level 5: Labels (12-14pt Dark/Bold)   ← STRUCTURE
Level 6: Body Text (11pt Dark)        ← CONTENT
Level 7: Metadata (7-10pt Gray/Gold)  ← DETAILS
```

## 📏 Technical Specifications

### Page Setup:
```
Format:        A4 (210mm × 297mm)
Orientation:   Landscape (842pt × 595pt)
Margins:       40pt all sides (content area)
Border Frame:  Outer 20pt, Inner 35pt from edge
```

### Safe Zones:
```
Outer Border:  20-35pt from edge (avoid content here)
Content Area:  35-560pt (top), 35-807pt (left-right)
Footer Area:   540-560pt from top
```

### Print Considerations:
- High contrast text (dark on light)
- No gradients or complex fills (solid colors only)
- Line weights ≥1pt (visible when printed)
- Font sizes ≥7pt (readable when printed)
- Proper margins for trimming

## 🎨 Customization Guide

### Easy Customizations:
1. **Change Gold Color**:
   - Edit `colors.accent` and `colors.accentDark` at top of file
   - All gold elements update automatically

2. **Add Hotel Logo**:
   - Insert image in top medallion area (centerX, 80)
   - Resize to fit 40pt diameter circle

3. **Change Fonts**:
   - Replace 'Helvetica-Bold' with your brand font
   - Ensure fonts are registered with PDFKit

4. **Adjust Performance Card**:
   - Change column positions (col1X, col2X, col3X)
   - Modify card background color (colors.cream)
   - Add/remove data fields

5. **Add Signature Images**:
   - Insert images above signature lines
   - Position at leftSigX and rightSigX coordinates

### Advanced Customizations:
1. Add watermarks (hotel logo faded in background)
2. Add embossed seal graphics
3. Insert QR code for verification
4. Add security features (microtext, patterns)
5. Multi-language support (bilingual certificate)
