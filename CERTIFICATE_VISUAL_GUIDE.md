## 🌟 Special Design Features

### Double Border Frame
- Creates luxury appearance
- Outer: 4pt gold, rounded corners (15pt radius)
- Inner: 1pt dark gold, rounded corners (8pt radius)
- Space between borders: 15pt

### Top & Bottom Medallions
- Center circle: 25pt radius
- Inner circle: 20pt radius (creates double-ring effect)
- Accent lines: 3pt thick, extend with 80pt gap from circle
- Perfect space for hotel logo

### Student Name Ornament
- 250pt long underline
- 2.5pt thick gold line
- Two 3pt gold circles, 8pt away from line ends
- Creates elegant framing effect

### Performance Card Accents
- Left gold bar: 6pt wide × 85pt tall
- Right gold bar: 6pt wide × 85pt tall
- Creates visual bookends for data
- Ties card to overall gold theme

## 📏 Spacing & Layout

### Page Structure:
```
Page: 842pt wide × 595pt tall (A4 Landscape)

┌─────────────────────────────────────┐
│ 20pt margin                          │
│  ┌─────────────────────────────┐    │
│  │ 35pt content margin          │    │
│  │                              │    │
│  │   [CERTIFICATE CONTENT]      │    │
│  │   595pt - 70pt = 525pt      │    │
│  │   content height            │    │
│  │                              │    │
│  └─────────────────────────────┘    │
│                                      │
└─────────────────────────────────────┘
```

### Vertical Spacing:
- Top medallion: 80pt from top
- Hotel name: 115pt from top
- Title: 195pt from top
- Student name: 275pt from top
- Performance card: 440pt from top
- Signatures: 110pt from bottom
- Footer: 55pt from bottom
- Bottom medallion: 28pt from bottom

### Horizontal Layout:
- Content centered with 35-40pt side margins
- Performance card: 110pt from left/right edges
- Signature positions: 200pt (left), 502pt (right)
- Centered elements use pageWidth/2 (421pt)
